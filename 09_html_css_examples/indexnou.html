1. Pagină statică simplă

O pagină statică este cea mai simplă formă de document HTML — conține text, imagini, titluri și paragrafe, dar nu are elemente dinamice (adică nu se modifică în funcție de acțiunile utilizatorului).

Acest tip de pagină este ideal pentru site-uri de prezentare, portofolii, articole informative sau pagini de tip „Despre noi”.

Exemplu practic:

Cod HTML pentru o pagină statică simplă:

<!DOCTYPE html>
<html lang="ro">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pagina mea statică</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <header>
      <h1>Bine ai venit pe site-ul meu!</h1>
    </header>

    <main>
      <p>
        Aceasta este o pagină statică simplă creată cu HTML și CSS.
        Pagina conține un titlu, un paragraf și o imagine reprezentativă.
      </p>

      <img
        src="https://via.placeholder.com/600x300"
        alt="Peisaj frumos"
        style="width:100%; border-radius:8px;"
      >

      <p>
        Folosind HTML putem structura conținutul, iar CSS ne ajută să
        îl stilizăm și să obținem o prezentare estetică și clară.
      </p>
    </main>

    <footer>
      <p>© 2025 – Pagina statică exemplu</p>
    </footer>
  </body>
</html>

Rezultatul vizual:

Aceasta este o pagină HTML simplă care conține text, imagini și stiluri CSS de bază. Tot conținutul este fix, adică nu se schimbă decât dacă edităm manual codul sursă.

Explicații:

  • <header> – zona de titlu a paginii;
  • <main> – conținutul principal al paginii;
  • <footer> – zona de jos a paginii, unde se pun drepturile de autor;
  • <img> – afișează o imagine;
  • <link rel="stylesheet"> – leagă fișierul CSS extern de stilizare.

Exercițiu propus:

Creează o pagină statică pentru un restaurant imaginar care să conțină:

  1. Un titlu cu numele restaurantului.
  2. O imagine reprezentativă.
  3. Un paragraf cu o scurtă descriere.
  4. Un footer cu adresa și anul curent.

Scopul exercițiului: să exersezi structura HTML și stilizarea simplă cu CSS.

2. Layout complex (Grid)

În dezvoltarea web modernă, CSS Grid este una dintre cele mai puternice metode de a crea un layout complex – adică o pagină împărțită în rânduri și coloane flexibile, care se adaptează automat la dimensiunea ecranului.

Prin Grid putem aranja elementele exact ca într-un tabel, dar cu mult mai mult control vizual și libertate de design.

Exemplu practic:

Cod HTML pentru un layout simplu cu trei coloane:

<!DOCTYPE html>
<html lang="ro">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Layout complex (Grid)</title>
    <link rel="stylesheet" href="style.css">
    <style>
      .grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        gap: 20px;
        padding: 20px;
      }

      .grid-item {
        background-color: #f3f3f3;
        border: 2px solid #ccc;
        padding: 20px;
        text-align: center;
        border-radius: 8px;
        font-weight: bold;
      }

      /* Responsivitate: pe ecrane mici, coloanele devin una sub alta */
      @media (max-width: 768px) {
        .grid-container {
          grid-template-columns: 1fr;
        }
      }
    </style>
  </head>
  <body>
    <header>
      <h1>Exemplu de Layout cu Grid</h1>
    </header>

    <main>
      <div class="grid-container">
        <div class="grid-item">Coloana 1</div>
        <div class="grid-item">Coloana 2</div>
        <div class="grid-item">Coloana 3</div>
      </div>
    </main>

    <footer>
      <p>© 2025 – Layout Grid exemplu</p>
    </footer>
  </body>
</html>

Rezultatul vizual:

Coloana 1
Coloana 2
Coloana 3

Explicații:

  • display: grid; – transformă containerul într-un grid (grilă);
  • grid-template-columns: repeat(3, 1fr); – creează trei coloane egale;
  • gap: 20px; – adaugă spațiu între coloane;
  • @media (max-width: 768px) – face layoutul responsiv pe telefoane;
  • .grid-item – fiecare „celulă” a grilei (coloană).

Exercițiu propus:

Creează un layout pentru o pagină de prezentare folosind CSS Grid, care să conțină:

  1. Un header cu titlul paginii;
  2. Trei coloane în zona principală (ex: „Despre noi”, „Servicii”, „Contact”);
  3. Un footer cu drepturile de autor.

Scopul exercițiului: să înveți cum poți organiza conținutul în coloane folosind CSS Grid și cum să îl faci adaptabil pe ecrane mici.

4. Carduri de produs

Cardurile de produs sunt elemente foarte folosite în paginile de comerț online. Ele afișează rapid informațiile esențiale despre un produs: imaginea, numele, descrierea și butonul de cumpărare.

Exemplu practic:

Codul este împărțit în două părți: HTML (structura) și CSS (designul).

🧱 Cod HTML

<div class="product-grid">
  <div class="product-card">
    <img src="https://via.placeholder.com/150" alt="Produs 1">
    <h3>Produs 1</h3>
    <p>Descriere scurtă a produsului.</p>
    <button>Cumpără</button>
  </div>

  <div class="product-card">
    <img src="https://via.placeholder.com/150" alt="Produs 2">
    <h3>Produs 2</h3>
    <p>Descriere scurtă a produsului.</p>
    <button>Cumpără</button>
  </div>
</div>

🎨 Cod CSS

.product-grid {
  display: flex;
  gap: 20px;
  justify-content: center;
  flex-wrap: wrap;
  margin: 30px 0;
}

.product-card {
  background-color: white;
  border: 1px solid #ddd;
  border-radius: 10px;
  width: 200px;
  padding: 15px;
  text-align: center;
  box-shadow: 0 4px 10px rgba(0,0,0,0.1);
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}

.product-card:hover {
  transform: translateY(-5px);
  box-shadow: 0 6px 14px rgba(0,0,0,0.15);
}

.product-card img {
  width: 100%;
  border-radius: 6px;
  margin-bottom: 10px;
}

.product-card h3 {
  margin-bottom: 8px;
  color: #20232a;
}

.product-card p {
  font-size: 0.9rem;
  color: #555;
  margin-bottom: 12px;
}

.product-card button {
  background-color: #61dafb;
  border: none;
  color: #20232a;
  padding: 8px 12px;
  border-radius: 6px;
  font-weight: bold;
  cursor: pointer;
  transition: background-color 0.2s ease;
}

.product-card button:hover {
  background-color: #4fc3f7;
}

Rezultatul vizual:

Produs 1

Descriere scurtă a produsului.

Produs 2

Descriere scurtă a produsului.

Explicații:

  • .product-grid – containerul principal, aranjează cardurile pe orizontală folosind Flexbox;
  • .product-card – fiecare card individual cu imagine, titlu, descriere și buton;
  • box-shadow – adaugă o umbră subtilă pentru efect de adâncime;
  • transition – creează o animație lină la hover;
  • flex-wrap: wrap; – permite ca elementele să treacă pe rândul următor pe ecrane mici.

Exercițiu propus:

Creează o pagină cu cel puțin 3 carduri de produse care să conțină:

  1. O imagine pentru fiecare produs;
  2. Titlu și descriere scurtă;
  3. Buton de cumpărare;
  4. Un efect vizual diferit la hover (ex: culoare de fundal schimbată sau mărire ușoară a imaginii).

Scopul exercițiului: să înveți cum se creează carduri moderne, responsive și atractive pentru prezentarea produselor pe un site web.

6. Galerie foto (Grid)

O galerie foto este un exemplu perfect de folosire a CSS Grid. Aceasta permite aranjarea imaginilor într-o grilă modernă, flexibilă și responsivă.

Exemplu practic:

Codul este împărțit în două secțiuni: HTML (structura) și CSS (stilizarea).

🧱 Cod HTML

<div class="gallery-grid">
  <img src="https://picsum.photos/300?random=1" alt="img1">
  <img src="https://picsum.photos/300?random=2" alt="img2">
  <img src="https://picsum.photos/300?random=3" alt="img3">
  <img src="https://picsum.photos/300?random=4" alt="img4">
</div>

🎨 Cod CSS

.gallery-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 15px;
  padding: 20px;
}

.gallery-grid img {
  width: 100%;
  height: 200px;
  object-fit: cover;
  border-radius: 10px;
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.gallery-grid img:hover {
  transform: scale(1.05);
  box-shadow: 0 6px 15px rgba(0, 0, 0, 0.3);
}

Rezultatul vizual:

Explicații:

  • display: grid; – transformă containerul într-o grilă;
  • grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); – creează coloane care se adaptează automat la ecran;
  • gap: 15px; – spațiu între imagini;
  • object-fit: cover; – păstrează proporțiile imaginilor fără deformare;
  • hover – adaugă un efect vizual la trecerea cursorului peste imagine.

Exercițiu propus:

Creează o galerie de 8 imagini care:

  1. Se aliniază automat pe 2–4 coloane, în funcție de dimensiunea ecranului;
  2. Are efect de mărire la hover (zoom in);
  3. Are un titlu sub fiecare imagine (folosește <figure> și <figcaption>).

Scopul exercițiului: să înveți cum poți crea o galerie de imagini elegantă și responsive folosind doar HTML și CSS Grid.

8. Template blog

Un template de blog simplu permite afișarea articolelor cu titlu, dată, imagine și text. Poate fi extins pentru mai multe articole.

🧱 Cod HTML

<div class="blog-container">

  <article class="blog-post">
    <h3>Primul articol</h3>
    <small>Publicat pe 22 octombrie 2025</small>
    <img src="https://picsum.photos/600/200?random=1" alt="Articol 1">
    <p>Acesta este un exemplu de articol de blog cu HTML și CSS. Poți adăuga titlu, text, dată și imagine.</p>
  </article>

  <article class="blog-post">
    <h3>Al doilea articol</h3>
    <small>Publicat pe 21 octombrie 2025</small>
    <img src="https://picsum.photos/600/200?random=2" alt="Articol 2">
    <p>Articolele pot fi stilizate diferit pentru a evidenția conținutul relevant.</p>
  </article>

</div>

🎨 Cod CSS

.blog-container {
  display: flex;
  flex-direction: column;
  gap: 20px;
  padding: 20px;
  max-width: 800px;
  margin: 0 auto;
}

.blog-post {
  background-color: #fff;
  padding: 15px;
  border-radius: 10px;
  box-shadow: 0 4px 10px rgba(0,0,0,0.1);
}

.blog-post h3 {
  margin-bottom: 5px;
  color: #20232a;
}

.blog-post small {
  display: block;
  margin-bottom: 10px;
  color: #555;
}

.blog-post img {
  width: 100%;
  border-radius: 8px;
  margin-bottom: 10px;
}

.blog-post p {
  line-height: 1.6;
  color: #333;
}

Rezultatul vizual:

Explicații:

  • Fiecare articol este încadrat în <article> pentru semnificație semantică;
  • Containerul .blog-container folosește Flexbox pentru spațiere verticală;
  • Imaginile au border-radius și width:100% pentru aspect uniform;
  • Box-shadow și padding oferă efect vizual de „card” pentru fiecare post.

Exercițiu propus:

  1. Adaugă un al treilea articol cu imagine și dată;
  2. Schimbă culorile fundalului și ale titlurilor pentru stil personal;
  3. Transformă articolele într-un grid cu două coloane pe ecrane mari (@media queries).

9. Pagină de tip CV

Un template simplu pentru CV sau Resume Page. Poți include date personale, experiență, educație și contact.

🧱 Cod HTML

<div class="resume-container">
  <header class="resume-header">
    <h1>Maria Popescu</h1>
    <h2>Web Developer</h2>
  </header>

  <section class="resume-section">
    <h3>Experiență</h3>
    <ul>
      <li>2022 - prezent: Freelancer Web Design</li>
      <li>2020 - 2022: Front-End Developer la ABC Studio</li>
    </ul>
  </section>

  <section class="resume-section">
    <h3>Educație</h3>
    <ul>
      <li>Facultatea de Informatică, București</li>
    </ul>
  </section>
</div>

🎨 Cod CSS

.resume-container {
  max-width: 600px;
  margin: 20px auto;
  padding: 20px;
  background-color: #fff;
  border-radius: 10px;
  box-shadow: 0 4px 10px rgba(0,0,0,0.1);
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

.resume-header h1 {
  margin-bottom: 5px;
  color: #20232a;
}

.resume-header h2 {
  margin-bottom: 15px;
  color: #555;
  font-weight: 400;
}

.resume-section {
  margin-bottom: 20px;
}

.resume-section h3 {
  margin-bottom: 10px;
  color: #20232a;
}

.resume-section ul {
  list-style: none;
  padding-left: 0;
}

  padding: 5px 0;
  border-left: 3px solid #61dafb;
  padding-left: 10px;
  margin-bottom: 5px;

Exercițiu propus:

  1. Adaugă o secțiune „Abilități” cu liste sau pictograme;
