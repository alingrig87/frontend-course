CSS Introduction

Fundamentele limbajului de stilizare pentru web: de la selectori și cascade la box model și layout.

Ce este CSS?

CSS (Cascading Style Sheets) descrie prezentarea documentelor HTML. Controlează culori, fonturi, spațieri, poziționări și răspunsul la diferite dimensiuni de ecran. 

/* CSS în fișier separat */
body { font-family: system-ui, -apple-system, Segoe UI, Roboto, sans-serif; }
/* CSS în <style> */
<style> h1 { color: #20232a; } </style>
/* CSS inline (de evitat) */
<h1 style="color:#20232a">Titlu</h1>
Principiul de bază: separă stilul de structură (HTML) pentru claritate și mentenanță.

Selectori, Specificitate & Cascadă

CSS aplică stiluri în funcție de selectori, iar conflictele se rezolvă prin specificitate și ordinea în cascadă. 

/* Selectori de element, clasă, ID */
h1 { font-size: 2rem; }
.card { border-radius: 8px; }
#highlight { background: #61dafb; }
/* Combinatori */
nav a { text-decoration: none; }
.list > li { margin-bottom: .5rem; }
/* Pseudoclase & pseudoelemente */
button:hover { transform: translateY(-1px); }
p::first-line { font-weight: 600; }

Box Model

Fiecare element este o cutie compusă din: content, padding, border, margin. 

*, *::before, *::after { box-sizing: border-box; }
.card {
 padding: 1rem;
 border: 1px solid #ddd;
 margin: 1rem 0;
}

Unități & Dimensiuni

  • px — pixeli; absolut.
  • em — relativ la font-size-ul elementului părinte.
  • rem — relativ la font-size-ul rădăcinii (html).
  • % — relativ la dimensiunea containerului.
  • vw/vh — procent din viewport.
  • clamp(min, preferat, max) — dimensiuni fluide cu limite. 
:root { --space: clamp(.75rem, 1vw + .5rem, 1.25rem); }
h1 { font-size: clamp(1.5rem, 2vw + 1rem, 3rem); }

Întrebări de interviu

1. Ce este specificitatea și cum se calculează?

Fiecare selector are un scor: ID > clasă/atribut/ pseudoclasă > element/pseudoelement. CSS aplică regula cu scorul mai mare; la egalitate, contează ordinea în fișier (ultimul câștigă).

2. Diferența între em și rem?

em este relativ la font-size-ul elementului părinte; rem este relativ la html. rem e mai previzibil pentru scale tipografice.
