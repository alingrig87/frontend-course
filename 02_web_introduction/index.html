💡 Introducere în Web

Acest curs explică fundamentele Internetului, World Wide Web-ului și ale arhitecturii web moderne.

🌍 Internet

Internetul este o rețea globală de calculatoare care trimit pachete mici de date digitale între ele — un „sistem poștal foarte rapid”.

  • A apărut din ARPAnet în anii 1960.
  • Permite trimiterea de pagini web, emailuri, fișiere media etc.
  • Folosește suita de protocoale TCP/IP pentru comunicare.

📡 Suita de protocoale TCP/IP

Este o suită cu arhitectură pe straturi:

  • Application Layer: HTTP, HTTPS, FTP, SMTP.
  • Transport Layer: TCP (fiabil), UDP (rapid dar nefiabil).
  • Network Layer: IP – adrese și rutare.
  • Data Link Layer: ARP, PPP – comunicarea hardware-rețea.

🕸️ World Wide Web

WWW este un sistem de documente legate între ele prin hyperlinkuri.

  • HTML – limbajul documentelor web.
  • URI / URL – identificarea resurselor.
  • HTTP – protocolul de transfer.

🔗 URL

Un URL (Uniform Resource Locator) indică locația unei resurse web și cum se obține aceasta.

https://developer.mozilla.org/en-US/docs/Web/HTTP/Methods/GET?source=sidebar#Syntax
  • Schemă: https
  • Domeniu: developer.mozilla.org
  • Cale: /en-US/docs/Web/HTTP/Methods/GET
  • Query: source=sidebar
  • Fragment: Syntax

💻 Arhitectura Client-Server

Modelul în care un client trimite o cerere către un server, iar serverul trimite un răspuns înapoi.

  • Exemple: Browser → Web server, Aplicație mobilă → Server API.
  • Poate fi single process (standalone) sau multi-layer.

🌐 DNS

DNS (Domain Name System) este „agenda telefonică” a internetului: traduce nume de domenii în adrese IP.

📨 HTTP și 🔒 HTTPS

HTTP (HyperText Transfer Protocol) este protocolul de comunicare între client și server.

  • Model Request → Response
  • Stateless
  • Text-based

HTTPS = HTTP + SSL/TLS (criptare). Protejează confidențialitatea și integritatea datelor transmise.

🧭 Web Browsers

Un browser este o aplicație client care afișează conținutul web:

  • Chrome
  • Firefox
  • Edge
  • Safari
  • Opera

🧰 Developer Tools

Instrumente integrate în browsere pentru dezvoltatori:

  • Inspectare DOM și CSS
  • Console pentru debugging JS
  • Analiză Network și Performance
