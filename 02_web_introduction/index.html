📘 Ce este Web-ul?

Web-ul (World Wide Web) este o rețea de documente interconectate care pot fi accesate prin internet. Aceste documente sunt construite folosind HTML, CSS și JavaScript.

🌐 Arhitectura Client-Server

Un utilizator (clientul) trimite cereri HTTP către un server, iar serverul răspunde cu resurse (HTML, CSS, JS, imagini etc.).

📄 Rolul HTML, CSS și JavaScript

  • 🔤 HTML definește structura paginii web.
  • 🎨 CSS controlează stilul și prezentarea paginii.
  • ⚙️ JavaScript adaugă comportament dinamic și interactivitate.

📡 HTTP și HTTPS

HTTP (Hypertext Transfer Protocol) este protocolul de bază pentru comunicare între client și server. HTTPS este versiunea sa securizată, care folosește criptarea SSL/TLS.

🧠 DOM (Document Object Model)

DOM reprezintă structura ierarhică a elementelor HTML. Poate fi manipulat folosind JavaScript pentru a modifica conținutul sau stilul unei pagini.

🧩 Exerciții practice

1️⃣ Creează o pagină HTML simplă

Scrie o pagină HTML cu un titlu, o imagine și un paragraf despre tine.

<!DOCTYPE html>
<html>
<head>
  <title>Despre mine</title>
</head>
<body>
  <h1>Salut, sunt Alex!</h1>
  <img src="poza.jpg" alt="eu" />
  <p>Îmi place programarea și web designul.</p>
</body>
</html>

2️⃣ Creează un buton cu efect hover

Adaugă tranziții CSS pentru a schimba culoarea la hover.

.btn {
  background-color: #4b3f72;
  color: white;
  padding: 10px 20px;
  border-radius: 8px;
  transition: background 0.3s;
}
.btn:hover {
  background-color: #7b1fa2;
}

🚀 Mini-proiect: Pagina personală de prezentare

📋 Obiectiv: creează o pagină personală care să combine HTML, CSS și puțin JavaScript.

  1. Inițializează un proiect cu git init
  2. Creează fișierul index.html
  3. Adaugă o secțiune cu un titlu și o imagine
  4. Adaugă o listă cu hobby-urile tale
  5. Adaugă un buton care, la click, afișează un mesaj
  6. Stilizează totul cu CSS (culori, fonturi, margini)
  7. Fă butonul să schimbe culoarea la hover
  8. Adaugă un link către profilul tău GitHub
  9. Salvează și dă commit cu git add . și git commit -m "Pagina mea"
  10. Publică proiectul pe GitHub Pages
# Comenzi Git
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/user/pagina-mea.git
git push -u origin main
