Flexbox — Layout flexibil în CSS

Flexbox este un sistem modern de aranjare a elementelor în pagină, care oferă control total asupra aliniamentului și spațierii.

1. Ce este Flexbox?

Flexbox (Flexible Box Layout) este un model de layout introdus în CSS3, conceput pentru a facilita distribuirea spațiului între elemente și alinierea lor într-un container flexibil. Scopul este de a face layout-urile responsive mai simple și mai previzibile.

2. Conceptul de bază

  • Containerul flexibil — elementul părinte cu display: flex;
  • Itemii flexibili — elementele copil care se aliniază automat în funcție de regulile stabilite;
  • Poți controla direcția, spațiul, ordinea și alinierea elementelor fără a folosi float-uri sau position absolute.

3. Proprietăți principale

  • justify-content — aliniază elementele pe axa principală;
  • align-items — aliniază pe axa transversală;
  • flex-direction — definește direcția (row / column);
  • flex-wrap — controlează împachetarea elementelor;
  • align-content — gestionează alinierea mai multor rânduri de itemi.

5. Exerciții
