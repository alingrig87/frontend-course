🔰 CSS Properties

În această lecție parcurgem proprietățile CSS de bază și avansate, cu exerciții practice, soluții ascunse și întrebări de interviu.

🧠 Teorie & Exerciții (cu soluții ascunse)

1. 🎨 Culori și background

Teorie:

  • color – schimbă culoarea textului
  • background-color – setează fundalul
  • background-image – adaugă o imagine ca fundal
  • background-repeat / background-size – controlează modul în care imaginea de fundal se repetă sau se scalează

Exercițiu: Creează un paragraf cu text roșu și fundal galben. 

<p style="color:red; background-color:yellow;">Text colorat</p>

2. 🖋️ Fonturi & text

Teorie:

  • font-family – definește familia de fonturi pentru text (ex: Arial, Times, monospace)
  • font-size – dimensiunea textului
  • font-weight – grosimea textului
  • text-align – aliniază textul: stânga, centru, dreapta, justify
  • text-decoration – adaugă efecte textului, cum ar fi underline sau line-through
  • line-height – spațierea verticală între rânduri
  • letter-spacing / word-spacing – spațiere între litere sau cuvinte

Exercițiu: Creează un heading cu font Arial, bold și centrat. 

<h2 style="font-family:Arial; font-weight:bold; text-align:center;">Titlu</h2>

3. 📦 Box Model

Teorie:

  • width / height – dimensiunea conținutului elementului
  • padding – spațiul interior între conținut și bordură
  • border – definește lățimea, stilul și culoarea bordurii elementului
  • margin – spațiul exterior dintre element și alte elemente
  • box-sizing – controlează dacă padding-ul și border-ul sunt incluse în dimensiunea totală (content-box vs border-box)

Exercițiu: Creează un div cu border, padding 10px și margin 15px. 

<div style="border:1px solid black; padding:10px; margin:15px;">Box</div>

4. 🖌️ Borduri & colțuri

Teorie:

  • border – definește lățimea, stilul și culoarea bordurii
  • border-radius – rotunjește colțurile elementului
  • box-shadow – adaugă umbre pentru efect vizual și profunzime

Exercițiu: Creează un div cu border rotunjit și culoare albastră. 

<div style="border:2px solid blue; border-radius:10px;">Colț rotunjit</div>

5. 🌀 Margin & Padding

Teorie:

  • margin – spațiu exterior elementului, între acesta și alte elemente
  • padding – spațiu interior elementului, între conținut și bordură

Exercițiu: Creează două div-uri cu spacing diferit între ele. 

<div style="margin:10px; padding:5px; border:1px solid black;">Div1</div>
<div style="margin:20px; padding:10px; border:1px solid black;">Div2</div>

6. 📑 Display & Visibility

Teorie:

  • display – definește cum este afișat un element: block, inline, inline-block, none
  • visibility – controlează vizibilitatea: visible sau hidden (elementul ocupă spațiu chiar dacă e invizibil)

Exercițiu: Creează un element ascuns cu display:none. 

<p style="display:none;">Nu se vede</p>

7. 📌 Positioning

Teorie:

  • position – stabilește tipul de poziționare: static, relative, absolute, fixed, sticky
  • Permite mutarea elementelor în pagină și combinarea cu top, left, right, bottom

Exercițiu: Creează un div poziționat absolut în interiorul unui container relativ. 

<div style="position:relative; width:200px; height:100px; border:1px solid black;">
  <div style="position:absolute; top:10px; right:10px;">Absolut</div>
</div>

8. ⚡ Flexbox

Teorie:

  • display: flex – transformă un container într-un flex container
  • justify-content – aliniază elementele pe axa principală
  • align-items / align-content – aliniază elementele pe axa transversală
  • flex-wrap – permite elementelor să se înfășoare pe rânduri multiple

Exercițiu: Creează un container flex cu 3 elemente aliniate pe orizontală. 

<div style="display:flex; justify-content:space-around;">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

9. 🔲 Grid

Teorie:

  • display: grid – definește un container ca grid
  • grid-template-columns / grid-template-rows – definește structura coloanelor și rândurilor
  • gap – distanța dintre rânduri și coloane
  • Permite aranjarea elementelor în layout-uri complexe și responsive

Exercițiu: Creează un grid cu 2 coloane și 4 itemi. 

<div style="display:grid; grid-template-columns:1fr 1fr; gap:10px;">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>

10. ✨ Pseudo-classes & elements

Teorie:

  • Pseudo-clase (:hover, :focus, :active) – stilizează stările unui element
  • Pseudo-elemente (::before, ::after) – creează conținut fictiv pentru stilizare
  • Folosite pentru interactivitate și efecte vizuale fără a adăuga elemente în HTML

Exercițiu: Schimbă culoarea unui link la hover. 

<a href="#" style="color:black;" onmouseover="this.style.color='red';" onmouseout="this.style.color='black';">Link</a>

11. 🎬 Animations & transitions

Teorie:

  • transition – controlează trecerea lină între stări diferite ale unei proprietăți
  • transform – rotește, scalează, translatează sau înclină elemente
  • animation – permite definirea unor animații complexe cu keyframes

Exercițiu: Creează un div care își schimbă culoarea la hover cu tranziție. 

<div style="width:100px; height:100px; background:red; transition:background 0.5s;" onmouseover="this.style.background='blue'" onmouseout="this.style.background='red'"></div>

💬 15 Întrebări frecvente de interviu CSS

🎨 Care este diferența între inline, inline-block și block?

Răspuns: inline nu începe pe un rând nou și ocupă doar cât conținutul; block începe pe un rând nou și ocupă lățimea completă; inline-block combină comportamentul inline cu posibilitatea de a avea dimensiuni definite.

🖋️ Cum se diferențiază em și rem pentru dimensiunea fontului?

Răspuns: em este relativ la dimensiunea fontului elementului părinte; rem este relativ la dimensiunea fontului rădăcină (html).

📦 Ce este Box Model și care sunt componentele sale?

Răspuns: Box model reprezintă orice element HTML ca un "box" format din: content → padding → border → margin. Dimensiunea totală depinde de aceste componente și de box-sizing.

🌀 Care este diferența dintre margin și padding?

Răspuns: margin este spațiul exterior între element și alte elemente; padding este spațiul interior între conținut și border.

📌 Cum funcționează position: relative/absolute/fixed/sticky?

Răspuns: relative – poziționare relativă față de poziția normală; absolute – față de primul părinte cu position non-static; fixed – față de viewport; sticky – combină relative și fixed când se scroll-ează.

⚡ Ce este Flexbox și cum se aliniază elementele pe axa principală și transversală?

Răspuns: Flexbox este un model de layout pentru alinierea și distribuirea elementelor într-un container. - Axă principală: justify-content aliniază elementele pe axa principală. - Axă transversală: align-items aliniază elementele pe axa transversală.

🔲 Cum se creează un grid și ce înseamnă fr în grid-template-columns?

Răspuns: Grid se creează cu display: grid. fr reprezintă o unitate flexibilă care distribuie spațiul disponibil între coloane sau rânduri.

✨ Care este diferența între :hover, :active și :focus?

Răspuns: :hover – când mouse-ul este deasupra elementului; :active – când elementul este apăsat; :focus – când elementul este selectat/activ (ex. input).

🎬 Cum funcționează transition și @keyframes?

Răspuns: transition creează efecte animate între stări de proprietăți CSS; @keyframes definește animații complexe pe mai multe etape.

🖌️ Cum se stilizează bordurile și cum se aplică border-radius?

Răspuns: border-width, border-style, border-color definesc bordura. border-radius rotunjește colțurile elementului.

💻 Ce diferență este între visibility: hidden și display: none?

Răspuns: visibility: hidden ascunde elementul, dar păstrează spațiul; display: none îl scoate complet din fluxul documentului.

🔗 Cum se stilizează link-urile diferit la hover și active?

Răspuns: Se folosesc pseudo-clasele :hover și :active pentru a aplica stiluri diferite la stările respective.

🌈 Cum se aplică culori cu hex, rgb, rgba și hsl?

Răspuns: #RRGGBB – hex; rgb(r,g,b) – valori numerice; rgba(r,g,b,a) – culori cu transparență; hsl(h,s,l) – hue, saturation, lightness.

🧩 Cum se folosește z-index și contextul de stacking?

Răspuns: z-index definește ordinea elementelor pe axa Z; elementele cu position non-static creează un context de stacking.

💡 Care sunt cele mai bune practici pentru responsive design cu CSS?

Răspuns: Folosește unități relative, media queries, Flexbox/Grid și testează pe diferite dimensiuni de ecran pentru a asigura un layout adaptiv.

🧪 Mini-proiect: „Pagina de stilizare CSS”

Obiectiv: Creează o pagină web simplă unde aplici proprietăți CSS învățate în lecție.

  1. 📄 Creează structura HTML de bază (doctype, head, body).
  2. 🎨 Aplică o culoare de fundal pentru body folosind background-color.
  3. 🖋️ Schimbă culoarea textului unui paragraf cu color.
  4. 📏 Modifică dimensiunea fontului pentru titlu cu font-size.
  5. 🖌️ Setează font-family pentru întregul body.
  6. 📦 Creează un div cu border și padding.
  7. 🌀 Aplică margin și padding pentru cel puțin două elemente.
  8. 🖼️ Adaugă o imagine și folosește width sau height pentru dimensiuni.
  9. 🔗 Stilizează link-uri (a) pentru hover și normal.
  10. 📑 Creează o listă ordonată și una neordonată cu culori diferite pentru li.
  11. 🧩 Bonus: Folosește class și id pentru a aplica stiluri diferite.
  12. 💻 Bonus: Experimentează cu text-align și display.
💡 Hint: Refă folosind cardurile și clasele din template pentru secțiuni.
📝 Nota: Acest mini-proiect ajută să vezi toate proprietățile CSS în acțiune pe o pagină reală.
