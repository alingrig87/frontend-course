🔰 CSS Properties

În această lecție parcurgem proprietățile CSS de bază și avansate, cu exerciții practice, soluții ascunse și întrebări de interviu.

🧠 Teorie & Exerciții (cu soluții ascunse)

1. 🎨 Culori și background

Teorie: color schimbă culoarea textului; background-color setează fundalul.

Exercițiu: Creează un paragraf cu text roșu și fundal galben. 

<p style="color:red; background-color:yellow;">Text colorat</p>

2. 🖋️ Fonturi & text

Teorie: font-family, font-size, font-weight, text-align, text-decoration.

Exercițiu: Creează un heading cu font Arial, bold și centrat. 

<h2 style="font-family:Arial; font-weight:bold; text-align:center;">Titlu</h2>

3. 📦 Box Model

Teorie: margin, border, padding, width/height.

Exercițiu: Creează un div cu border, padding 10px și margin 15px. 

<div style="border:1px solid black; padding:10px; margin:15px;">Box</div>

4. 🖌️ Borduri & colțuri

Teorie: border-radius rotunjeste colțurile; border-style, border-color.

Exercițiu: Creează un div cu border rotunjit și culoare albastră. 

<div style="border:2px solid blue; border-radius:10px;">Colț rotunjit</div>

5. 🌀 Margin & Padding

Teorie: margin = spațiu exterior, padding = spațiu interior.

Exercițiu: Creează două div-uri cu spacing diferit între ele. 

<div style="margin:10px; padding:5px; border:1px solid black;">Div1</div>
<div style="margin:20px; padding:10px; border:1px solid black;">Div2</div>

6. 📑 Display & Visibility

Teorie: display: block/inline/inline-block/none, visibility: visible/hidden.

Exercițiu: Creează un element ascuns cu display:none. 

<p style="display:none;">Nu se vede</p>

7. 📌 Positioning

Teorie: static, relative, absolute, fixed, sticky.

Exercițiu: Creează un div poziționat absolut în interiorul unui container relativ. 

<div style="position:relative; width:200px; height:100px; border:1px solid black;">
  <div style="position:absolute; top:10px; right:10px;">Absolut</div>
</div>

8. ⚡ Flexbox

Teorie: display:flex, justify-content, align-items.

Exercițiu: Creează un container flex cu 3 elemente aliniate pe orizontală. 

<div style="display:flex; justify-content:space-around;">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

9. 🔲 Grid

Teorie: display:grid, grid-template-columns, gap.

Exercițiu: Creează un grid cu 2 coloane și 4 itemi. 

<div style="display:grid; grid-template-columns:1fr 1fr; gap:10px;">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>

10. ✨ Pseudo-classes & elements

Teorie: :hover, :first-child, ::before, ::after.

Exercițiu: Schimbă culoarea unui link la hover. 

<a href="#" style="color:black;" onmouseover="this.style.color='red';" onmouseout="this.style.color='black';">Link</a>

11. 🎬 Animations & transitions

Teorie: transition = tranziție între stări; @keyframes pentru animații.

Exercițiu: Creează un div care își schimbă culoarea la hover cu tranziție. 

<div style="width:100px; height:100px; background:red; transition:background 0.5s;" onmouseover="this.style.background='blue'" onmouseout="this.style.background='red'"></div>

🧪 Mini-proiect: „Pagina de stilizare CSS”

Obiectiv: Creează o pagină web simplă unde aplici proprietăți CSS învățate în lecție.

  1. 📄 Creează structura HTML de bază (doctype, head, body).
  2. 🎨 Aplică o culoare de fundal pentru body folosind background-color.
  3. 🖋️ Schimbă culoarea textului unui paragraf cu color.
  4. 📏 Modifică dimensiunea fontului pentru titlu cu font-size.
  5. 🖌️ Setează font-family pentru întregul body.
  6. 📦 Creează un div cu border și padding.
  7. 🌀 Aplică margin și padding pentru cel puțin două elemente.
  8. 🖼️ Adaugă o imagine și folosește width sau height pentru dimensiuni.
  9. 🔗 Stilizează link-uri (a) pentru hover și normal.
  10. 📑 Creează o listă ordonată și una neordonată cu culori diferite pentru li.
  11. 🧩 Bonus: Folosește class și id pentru a aplica stiluri diferite.
  12. 💻 Bonus: Experimentează cu text-align și display.
💡 Hint: Refă folosind cardurile și clasele din template pentru secțiuni.
📝 Nota: Acest mini-proiect ajută să vezi toate proprietățile CSS în acțiune pe o pagină reală.
