CSS Grid — Tutorial Complet

1. Introducere în CSS Grid

CSS Grid este un sistem modern de layout în CSS, care permite organizarea elementelor într-o grilă bidimensională (rânduri și coloane). Este extrem de util pentru designuri complexe și responsive. 


.container {
  display: grid;
}

2. Container vs Item

Într-un layout Grid, există două niveluri: containerul (elementul părinte) și itemii (copiii direcți). Proprietățile de layout se aplică diferit. 


.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

.grid-item {
  background: lightblue;
  padding: 1rem;
}

3. grid-template-columns / rows

Proprietățile grid-template-columns și grid-template-rows definesc structura grilei. 


.grid-container {
  display: grid;
  grid-template-columns: 200px 1fr 100px;
  grid-template-rows: auto auto;
}

4. gap, grid-area

gap controlează distanța dintre celulele grilei. grid-area definește poziția unui element. 


.grid-container {
  display: grid;
  gap: 10px;
}

.header {
  grid-area: header;
}

5. repeat() și fr units

Funcția repeat() și unitatea fr ajută la crearea grilelor flexibile și proporționale. 


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

6. Aliniere pe axele X și Y

justify-content și align-content controlează alinierea grilei în container. 


.grid-container {
  display: grid;
  justify-content: center;
  align-items: center;
}

7. Implicit grid placement

Dacă elementele nu au o poziție definită, Grid le plasează automat pe rânduri noi, în funcție de ordinea din HTML. 


.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

8. Grid + Flexbox combo

Poți combina Grid (pentru structura principală) cu Flexbox (pentru alinierea internă a elementelor). 


.grid-item {
  display: flex;
  justify-content: center;
  align-items: center;
}

9. Workshop: Dashboard Layout

Exemplu practic: creare layout dashboard cu sidebar, header și zona de conținut. 


.dashboard {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: 60px 1fr;
  grid-template-areas:
    "sidebar header"
    "sidebar content";
}

10. Tema: Layout de pagină complet

Creează o pagină completă folosind CSS Grid: header, meniu, conținut, sidebar și footer. 


.page {
  display: grid;
  grid-template-areas:
    "header header"
    "menu content"
    "footer footer";
}
