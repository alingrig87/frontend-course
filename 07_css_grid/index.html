CSS Grid — Partea teoretică

În acest modul învățăm principiile CSS Grid: cum definim rânduri/coloane, cum plasăm elementele, și cum construim layout-uri responsiv.

📘 Sumar scurt:
  • display: grid; activează sistemul de grid pentru un container.
  • grid-template-columns și grid-template-rows definesc coloanele și rândurile.
  • grid-column, grid-row permit plasarea unui item pe mai multe piste.
  • gap controlează spațiul între celule.
  • minmax(), fr și auto-fit / auto-fill sunt utile pentru layout-uri fluide.

💬 Întrebări de interviu — CSS Grid (cât mai multe)

1. Ce face grid-template-columns: 1fr 2fr;?

👉 Creează două coloane; a doua are dublul spațiului față de prima (1fr vs 2fr).

2. Care este diferența între auto-fit și auto-fill?

👉 auto-fill umple rândul cu coloane chiar dacă sunt goale; auto-fit împachetează coloanele disponibile, trăgând în jos coloanele goale.

3. Când ai folosi minmax()?

👉 Când vrei să limitezi dimensiunea minimă și maximă a unei piste (ex: minmax(150px, 1fr)).

4. Cum aliniez conținutul din grid pe axa verticală?

👉 Folosește align-items sau align-self pentru itemi; align-content pentru spațiul dintre rânduri.

5. Cum fac un item să înceapă pe coloana 2 și să se termine pe coloana 4?

👉 grid-column: 2 / 4;

6. Ce proprietate folosești pentru spațiere între rânduri/coloane?

👉 gap (sau row-gap și column-gap).

7. Poți folosi grid pe elemente inline?

👉 Poți folosi display: inline-grid; dacă vrei comportament inline.

8. Cum inversezi ordinea itemilor în grid fără a schimba HTML-ul?

👉 Folosește order pe itemi sau redenumește explicit cu grid-area și grid-template-areas.

9. Ce înseamnă unitatea fr?

👉 fr este o fracțiune din spațiul disponibil al containerului grid.

10. Cum pot crea layout-uri complexe cu grid-template-areas?

👉 Definiți denumiri pentru zone în grid-template-areas și aplicați grid-area itemilor.

11. Ce se întâmplă dacă un item depășește zona definită?

👉 Itemul poate suprapune alte celule sau trunchia conținutul; gestionează cu overflow sau ajustează pistele.

12. Cum combini Grid cu Flexbox?

👉 Folosește Grid la nivel de layout (container mare) și Flexbox pentru alinieri interne (ex: butoane în carduri).
