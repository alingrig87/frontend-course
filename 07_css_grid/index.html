CSS Grid — Tutorial Practic

01. Grid Basics ▼

Grid-ul este un sistem bidimensional pentru rânduri și coloane. Se definește cu display: grid pe container și se configurează rândurile și coloanele. 

.container { display: grid; grid-template-columns: 1fr 1fr; gap: 10px; }

Exercițiul 2: Creează un grid cu 3 coloane și gap de 15px.

💬 Întrebări de interviu — CSS Grid ▼

1. Ce face grid-template-columns: 1fr 2fr;?

Creează două coloane; a doua are dublu spațiu față de prima.

2. Care este diferența între grid-gap și gap?

gap este standardul modern, grid-gap e vechi, doar pentru grid.

3. Ce înseamnă grid-auto-rows: minmax(100px, auto);?

Rândurile noi vor avea minim 100px și se vor extinde automat dacă conținutul e mai mare.

4. Cum aliniezi un element în centru folosind Grid?

Folosești justify-self: center; și align-self: center; sau place-self: center;.

5. Ce face grid-template-areas?

Permite să denumești zone ale gridului și să plasezi elemente prin nume, nu prin index numeric.

6. Care este diferența între auto-fill și auto-fit?

auto-fill păstrează spațiile goale, auto-fit expandează elementele ca să umple spațiul disponibil.

7. Cum faci ca un element să ocupe 2 coloane?

Folosești grid-column: span 2; sau grid-column: 1 / 3;.

8. Care e diferența între fr și %?

fr e unitate flexibilă pentru spațiu disponibil în grid, % depinde de containerul părinte.

9. Cum controlezi ordinea elementelor în grid?

Folosind proprietatea order pe elementele grid.

10. Ce face justify-items: stretch;?

Elementele se extind pe axa orizontală pentru a umple complet coloana lor.

11. Ce face align-items: start;?

Aliniază elementele la începutul rândului pe axa verticală.

12. Cum creezi o zonă centrală mai mare decât restul cu grid-template-areas?

Definind numele acelei zone pe mai multe rânduri și/sau coloane în șirurile de grid-template-areas.

13. Ce face grid-auto-flow: dense;?

Umple spațiile goale în grid cât mai compact posibil.

14. Cum setezi toate elementele să aibă aceeași înălțime?

Folosind align-items: stretch; pe containerul grid.

15. Cum faci un grid responsive cu minmax()?

Setezi dimensiunea minimă și maximă a coloanelor/rândurilor, de exemplu grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));

16. Ce face place-items: center;?

Aliniază elementele pe axa orizontală și verticală în centru.

17. Care e diferența între grid-template-rows și grid-auto-rows?

grid-template-rows definește rândurile explicite; grid-auto-rows definește rândurile create automat pentru conținut suplimentar.

18. Cum faci ca un element să ocupe mai multe rânduri?

Folosind grid-row: span 2; sau grid-row: 1 / 3;.

19. Ce face grid-area: header;?

Plasează elementul în zona definită cu grid-template-areas: "header ...";.

20. Ce proprietate folosești pentru a da spațiu între rânduri și coloane?

Folosind gap, row-gap și column-gap.
