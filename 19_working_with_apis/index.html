Working with APIs

În acest capitol vom explora cum funcționează API-urile și cum pot fi folosite în aplicațiile web moderne.

Ce este un API?

Un API (Application Programming Interface) este o interfață care permite aplicațiilor să comunice între ele. Practic, un API oferă un set de reguli și endpoint-uri prin care o aplicație poate cere sau trimite date către un server. 

GET https://api.exemplu.com/products

Răspunsul este, de obicei, în format JSON, un format text ușor de citit și prelucrat în JavaScript.
