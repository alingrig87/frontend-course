Structura de bază a unui document HTML

Orice pagină HTML are o structură de bază formată din câteva elemente principale: <!DOCTYPE>, <html>, <head> și <body>. 

<!DOCTYPE html>
<html lang="ro">
  <head>
    <meta charset="UTF-8">
    <title>Titlul paginii</title>
  </head>
  <body>
    <h1>Salut lume!</h1>
  </body>
</html>

Elementul <body> conține tot conținutul vizibil al paginii: texte, imagini, link-uri etc.
