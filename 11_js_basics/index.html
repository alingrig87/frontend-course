🧠 Bazele JavaScript

Învață conceptele fundamentale: variabile, funcții, bucle, evenimente și manipularea DOM-ului.

📘 Partea teoretică

1️⃣ Variabile: let și const sunt recomandate; var e mai vechi. 

let nume = "Xonia";
const PI = 3.14;

2️⃣ Tipuri de date: String, Number, Boolean, Array, Object. 

let text = "Salut!";
let numar = 42;
let adevar = true;
let lista = ["HTML", "CSS", "JS"];
let persoana = { nume: "Alina", varsta: 30 };

3️⃣ Funcții: Blocoane de cod reutilizabile.

function salut() {
  console.log("Bine ai venit!");
}
salut();

⚙️ Exerciții practice

1️⃣ Creează o funcție care calculează suma a două numere

function suma(a, b) {
  return a + b;
}
console.log(suma(5, 7));

2️⃣ Afișează toate elementele dintr-un array folosind o buclă

let limbaje = ["HTML", "CSS", "JS"];
for(let i = 0; i < limbaje.length; i++) {
  console.log(limbaje[i]);
}
