Capitolul 2: Media Queries

Media Queries sunt reguli CSS care permit aplicarea de stiluri diferite în funcție de caracteristicile dispozitivului (lățime, înălțime, rezoluție etc.). Ele sunt cheia pentru design responsive, deoarece permit adaptarea layout-ului la orice ecran.

Exemplu practic:

Trei coloane pe desktop devin una sub alta pe tablete și mobile folosind media queries.

Coloana 1
Coloana 2
Coloana 3
.media-container {
  display: flex;
  gap: 1rem;
}

.box {
  flex: 1;
  background-color: #ff6b6b;
  color: white;
  padding: 1rem;
  text-align: center;
  border-radius: 8px;
}

/* Media Queries: pe tablete și mobile coloanele se așază vertical */
@media (max-width: 1024px) {
  .media-container {
    flex-direction: column;
  }
}

Puncte cheie:

  • Media Queries permit stiluri diferite pentru diferite dimensiuni de ecran.
  • Se pot aplica pentru layout, fonturi, imagini și alte elemente.
  • Cheia pentru design responsive.
  • Se folosesc frecvent împreună cu unități relative și flex/grid.
