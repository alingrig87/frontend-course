Capitolul 7: Mobile-first design

Mobile-first design este o abordare unde proiectăm inițial pentru dispozitive mobile și apoi adaptăm layout-ul pentru ecrane mai mari folosind media queries. Avantaje:

  • Îmbunătățește experiența utilizatorilor de pe mobil
  • Încărcare mai rapidă pe dispozitive mobile
  • Facilitează design-ul responsive complet

Exemplu practic:

Pe mobil coloanele sunt verticale, iar pe desktop devin orizontale.

/* Container Mobile-first */
.mobile-first-container {
  display: flex;
  flex-direction: column; /* mobil: vertical */
  gap: 1rem;
}

.mobile-first-box {
  background-color: #1982c4;
  color: white;
  padding: 1rem;
  border-radius: 8px;
  text-align: center;
  flex: 1;
}

@media (min-width: 768px) {
  .mobile-first-container {
    flex-direction: row; /* desktop: orizontal */
  }
}

Puncte cheie:

  • Proiectarea începe pentru mobil și apoi se adaptează la desktop.
  • Media queries cu min-width schimbă layout-ul pe ecrane mari.
  • Asigură o experiență optimă pentru toate dispozitivele.
