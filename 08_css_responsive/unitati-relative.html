Capitolul 5: Unități relative

Unitățile relative permit ca elementele să se ajusteze automat la dimensiunea ecranului sau la mărimea fontului de bază. Cele mai folosite sunt:

  • % – procent din container
  • vw – procent din lățimea viewport-ului
  • vh – procent din înălțimea viewport-ului
  • rem – dimensiune relativă la font-size-ul rădăcină

Exemplu practic:

30% width
20vw width
20vh height
2rem font-size
/* Box cu lățime procentuală */
.box-percent { width: 30%; }

/* Box cu lățime relativă la viewport */
.box-vw { width: 20vw; }

/* Box cu înălțime relativă la viewport */
.box-vh { height: 20vh; }

/* Font-size relativ la root */
.box-rem { font-size: 2rem; }

@media (max-width: 768px) {
  .unit-container { flex-direction: column; }
}

Puncte cheie:

  • Unitățile relative fac site-ul flexibil și adaptabil.
  • % și vw/vh se folosesc pentru layout și dimensiuni de ecran.
  • rem este util pentru mărimea textului și spațiere consistentă.
  • Combinarea cu media queries oferă un design complet responsive.
