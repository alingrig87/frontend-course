Capitolul 4: Layout fluid vs fix

Layout fix are lățimi precise (pixeli) și nu se ajustează la dimensiunea ecranului. Layout fluid folosește unități relative (%, flex) și se adaptează automat la ecran.

Exemplu practic:

Coloana verde are lățime fixă, iar coloana albastră se ajustează automat la ecran.

Coloană fixă
Coloană fluidă
/* Fixă */
.box-fixed {
  width: 300px;
}

/* Fluidă */
.box-fluid {
  flex: 1;
}
@media (max-width: 768px) {
  .layout-container {
    flex-direction: column;
  }
}

Puncte cheie:

  • Layout fix: dimensiune constantă, nu se adaptează la ecran.
  • Layout fluid: folosește unități relative și se ajustează automat.
  • Pe mobil, coloanele se pot aranja vertical.
  • Layout fluid este recomandat pentru design responsive modern.
