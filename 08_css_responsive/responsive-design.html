Capitolul 1: Ce este Responsive Design

Responsive Design înseamnă crearea de site-uri care se adaptează automat la dimensiunea ecranului dispozitivului pe care sunt vizualizate, fie că este desktop, tabletă sau mobil. Scopul este ca utilizatorul să aibă o experiență optimă indiferent de device.

Exemplu practic:

Două coloane pe desktop devin una sub alta pe mobil.

Coloana 1
Coloana 2
.responsive-container {
  display: flex;
  gap: 1rem;
}

.box {
  flex: 1;
  background-color: #00b4d8;
  color: white;
  padding: 1rem;
  text-align: center;
  border-radius: 8px;
}

/* Pe ecrane mici, coloanele se așază una sub alta */
@media (max-width: 768px) {
  .responsive-container {
    flex-direction: column;
  }
}

Puncte cheie:

  • Layout-ul se adaptează la orice dimensiune de ecran.
  • Imaginile și textele sunt scalabile.
  • Experiență consistentă pentru utilizator.
  • Se folosesc unități relative și media queries pentru ajustări.
