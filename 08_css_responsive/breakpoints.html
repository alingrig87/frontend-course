Capitolul 3: Breakpoints standard

Breakpoints sunt lățimile de ecran la care design-ul site-ului trebuie să se adapteze. Acestea sunt praguri standard folosite pentru a defini stiluri diferite în funcție de dimensiunea dispozitivului.

Exemplu practic:

Trei coloane pe desktop și tabletă care se transformă vertical pe mobil folosind breakpoints standard.

Coloana 1
Coloana 2
Coloana 3
.breakpoint-container {
  display: flex;
  gap: 1rem;
}

.box {
  flex: 1;
  background-color: #ffb703;
  color: white;
  padding: 1rem;
  text-align: center;
  border-radius: 8px;
}

/* Breakpoints standard */
@media (max-width: 1200px) { /* Large screens */ }
@media (max-width: 992px)  { /* Medium screens */ }
@media (max-width: 768px)  { /* Small screens - coloane verticale */ 
  .breakpoint-container {
    flex-direction: column;
  }
}

Puncte cheie:

  • Breakpoints definesc când layout-ul trebuie ajustat.
  • Standard: 1200px (desktop mare), 992px (tabletă), 768px (mobil).
  • Se folosesc cu media queries pentru design responsive.
  • Permite o experiență consistentă pe toate dispozitivele.
