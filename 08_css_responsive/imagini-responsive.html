Capitolul 6: Imagini responsive

Imaginile responsive se ajustează automat la dimensiunea ecranului, păstrând proporțiile și integritatea designului. Cel mai simplu mod de a le face responsive este folosind:

  • width: 100% – imaginea se întinde până la lățimea containerului
  • height: auto – păstrează proporțiile
  • max-width – limitează dimensiunea maximă pe ecrane mari

Exemplu practic:

.image-container img {
  width: 100%;       /* se întinde până la container */
  max-width: 300px;  /* limitează dimensiunea maximă */
  height: auto;      /* păstrează proporțiile */
  border-radius: 8px;
  box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

@media (max-width: 768px) {
  .image-container {
    flex-direction: column;
    align-items: center;
  }
}

Puncte cheie:

  • Imaginile responsive se ajustează automat la dimensiunea ecranului.
  • Folosește width: 100% și height: auto pentru adaptabilitate.
  • Max-width previne imaginile prea mari pe ecrane mari.
  • Combină cu flex sau grid pentru layout responsive complet.
