Ce este Frontend Development

Frontend development reprezintă partea vizibilă a unui website — tot ceea ce utilizatorul vede și cu ce interacționează direct în browser. Un frontend developer folosește tehnologii ca HTML, CSS și JavaScript pentru a crea interfețe moderne, interactive și responsive.

Un proiect frontend tipic se bazează pe un mediu de dezvoltare local cu instrumente precum:

  • Git — pentru versionare;
  • VS Code — editor de cod;
  • Node.js & npm — pentru gestionarea pachetelor și rularea scripturilor;
  • CLI — interfața de comandă pentru automatizări și execuții rapide.

În această secțiune, vei învăța cum să configurezi mediul de lucru complet pentru a putea dezvolta proiecte frontend moderne. 

# Exemplu: verificare versiune Node.js și npm
node -v
npm -v

# Inițializare proiect nou
npm init -y

Pe măsură ce avansezi, vei folosi GitHub pentru a publica proiectele tale și pentru a colabora cu alți dezvoltatori folosind un flux profesional bazat pe branching și pull requests.
