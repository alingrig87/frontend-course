💬 Întrebări de interviu

1. Ce este diferența dintre let, const și var?

👉 var are scop de funcție, let și const au scop de bloc; const nu poate fi reatribuit.

2. Ce este DOM și cum îl putem manipula?

👉 DOM este o reprezentare obiectuală a structurii HTML; putem manipula nodurile cu document.querySelector() și metodele asociate.
